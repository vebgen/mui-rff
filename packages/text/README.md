# text

A text component that you can use in a React-Final-Form.

As with the native text input, it can take various forma:

- password,
- text,
- email,
- number,
- url,
- tel,
- date,
- datetime-local,
- month,
- time,
- week,
- color.

## Running unit tests

Run `pnpm nx test text` to execute the unit tests via
[Jest](https://jestjs.io).
